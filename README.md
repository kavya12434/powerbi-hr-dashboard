# Power BI HR Dashboard

The dataset contains HR employee information such as salary, department, and performance.

File used:

* hr-data.xlsx

* Power BI
* Microsoft Excel

## Files

* hr-dashbord.pbix → Power BI dashboard

## Insights

* Salary distribution
* Department analysis
* Employee performance
